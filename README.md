# DEVTA-AI-chatbot-for-my-personal-pc-
Devta is a Windows-native Jarvis-style AI assistant. It listens continuously with an always-on microphone, activates on wake words ("Hello Devta" / "Bhai Devta"), responds in voice and text, controls your PC, and proactively sends smart notification suggestions while you work or browse. Say "Devta stop" to silence suggestions.
